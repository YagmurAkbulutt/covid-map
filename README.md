## Covid-Map
- Bu proje, kullanıcıların bir dünya haritası üzerinden ülke seçerek o ülkedeki COVID-19 vakalarını görüntülemelerini sağlayan React tabanlı bir web uygulamasıdır. Uygulama, sorunsuz bir kullanıcı deneyimi sağlamak için çeşitli kütüphaneler ve araçlar kullanmaktadır.

## Özellikler
- Kullanıcı dostu arayüz
- Dünya haritası üzerinden ülke seçimi
- Seçilen ülkedeki COVID-19 vaka sayısını görüntüleme
- Gerçek zamanlı veri güncellemeleri

## Kullanılan Teknolojiler
- React: Kullanıcı arayüzü için
- Redux: Durum yönetimi için
- Redux Thunk: Asenkron işlemler için
- React Router: Uygulama içinde gezinme için
- Axios: HTTP istekleri için
- React Simple Maps: Dünya haritası gösterimi için
- React Icons: İkonlar için
- Testing Library: Testler için
